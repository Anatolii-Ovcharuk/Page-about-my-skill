Instruction for use - Инструкция для пользования

Instruction for use:

1. For close section after view - "Click" on button this is section again.

2. If page is not working or not view correctly - Open this page in other browsers.

3. If page is broken or not working of something - "Click" on button "Refresh page" (or use "Ctrl+R").

Инструкция для пользования:

1. Для закрытия секции после просмотра - Нажмите на кнопку этой секции снова.

2. Если страница не работает или отображается некоректно - Откройте страницу в других браузерах.

3. Если страница "сломана" или что-то не работает - Нажмите на кнопку "Обновить страницу" (или используйте комбинацию клавиш "Ctrl+R").
